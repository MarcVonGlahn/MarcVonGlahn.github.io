Master Thesis — UE5 Plugin

This page is still under construction.

My Master's Thesis, titled "Contextual Augmentation of Motion Capture Animations using Procedural Animation Techniques and Parameterization", resulted in a custom plugin for Unreal Engine 5. The plugin enables animators and developers to tweak parameters that procedurally modify existing motion capture animations — transforming a simple walk cycle into an injured limp, a drunken sway, or other contextual variations, all without requiring additional mocap sessions.

Under the hood, the system leverages various procedural animation principles, including inverse kinematics, to blend and parameterize motion data in real-time. The goal is to give artists a flexible tool that bridges the gap between handcrafted mocap data and the need for contextual variation in game characters.

Thanks to Rokoko, we could also make use of a Motion Capture suit to further understand the technology and incorportate custom animations into our project.

Key Features

  • Custom UE5 plugin for procedural animation modification
  • Parameterize motion capture data to create contextual variations (injured limp, drunken sway, etc.)
  • Procedural animation techniques including inverse kinematics
  • Master's Thesis project at IT University Copenhagen
