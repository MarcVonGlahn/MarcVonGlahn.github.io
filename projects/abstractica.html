ABSTRACTICA

Third Person Shooter with Roguelike Elements

This page is still under construction.

This is a third person shooter with roguelike elements, devoloped by myself in Unreal Engine 5. The project started as a learning experience for myself to deepen my knowledge in UE5, and then naturally progressed into a full game that is available to wishlist on Steam.

I am responsible for every aspect of the game, but on this page I want to showcase what I did in regards to 4 areas: Gameplay Programming, Shader Programming, Game AI, and the use of the Gameplay Ability System. In developement, I leverage a mix of C++ and Blueprints, trying to implement as much of the heavy lifting in C++ as possible. But more on that below.

Key Features

  • Third person shooter with roguelike elements and permanent progression
  • Highly stylized world with no lit material using custom shaders
  • Gameplay Ability System for skill and ability management
  • Multi-phased boss enemy design
  • Expanding color change shader using Signed Distance Fields

// Grant the ability to the ASC at begin play
void AAbstracticaCharacter::BeginPlay()
{
    Super::BeginPlay();

    if (HasAuthority() && AbilitySystemComponent)
    {
        for (TSubclassOf<UGameplayAbility>& Ability : DefaultAbilities)
        {
            AbilitySystemComponent->GiveAbility(
                FGameplayAbilitySpec(Ability, 1, INDEX_NONE, this));
        }
    }
}

// Activate by gameplay tag (called from input binding)
void AAbstracticaCharacter::ActivatePrimaryAbility()
{
    FGameplayTagContainer TagContainer;
    TagContainer.AddTag(FGameplayTag::RequestGameplayTag(FName("Ability.Primary")));
    AbilitySystemComponent->TryActivateAbilitiesByTag(TagContainer);
}
